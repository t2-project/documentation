public void addItemToCart(String sessionId, String productId, Integer units)¶

Add the given number units of product to a users cart.

If the product is already in the cart, the units of that product will be updated.

Parameters:
  • sessionId – identifies the cart to add to
  • productId – id of product to be added
  • units – number of units to be added
Throws:
  • CartInteractionFailedException – if the request number of unit could not be placed in the cart.

deleteItemFromCart¶

public void deleteItemFromCart(String sessionId, String productId, Integer units)¶

Delete the given number units of product from a users cart.

If the number of units in the cart decrease to zero or less, the product is remove from the cart. If the no such product is in cart, do nothing.

Parameters:
  • sessionId – identifies the cart to delete from
  • productId – id of the product to be deleted
  • units – number of units to be deleted
Throws:
  • CartInteractionFailedException – if anything went wrong while talking to the cart
