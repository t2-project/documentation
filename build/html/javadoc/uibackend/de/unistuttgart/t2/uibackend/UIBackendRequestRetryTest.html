UIBackendRequestRetryTest¶

public class UIBackendRequestRetryTest¶

Test whether UIBackendservice retries requests as it should. The set up is similar to UIBackendRequestTest except that the mock server does now always reply with a HttpStatus.INTERNAL_SERVER_ERROR at some point and that the the test always assert that the request that received the Error is placed twice.
