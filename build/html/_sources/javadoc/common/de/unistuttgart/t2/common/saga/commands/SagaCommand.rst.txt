SagaCommand
===========

.. java:package:: de.unistuttgart.t2.common.saga.commands
   :noindex:

.. java:type:: public abstract class SagaCommand implements Command

   Generic command to the saga participants.

   Holds the \ :java:ref:`SagaData <de.unistuttgart.t2.common.saga.SagaData>`\  that the saga participants need to perform actions or compensations.

   Storing the names of the queues here is not intended, but i do it anyway because i want them at one central place.

   :author: maumau
